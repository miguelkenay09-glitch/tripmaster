# Contributing to TripMaster\n\nObrigado por estar interessado em contribuir para o TripMaster! 🎉\n\n## Como Contribuir\n\n### 1. Fork o Repositório\n```bash\nfork: https://github.com/miguelkenay09-glitch/tripmaster\n```\n\n### 2. Clone Seu Fork\n```bash\ngit clone https://github.com/seu-usuario/tripmaster.git\ncd tripmaster\n```\n\n### 3. Crie uma Branch\n```bash\ngit checkout -b feature/sua-feature\n# ou\ngit checkout -b fix/seu-bug\n```\n\n### 4. Faça suas Mudanças\n- Siga o código style existente\n- Escreva testes para novas features\n- Atualize a documentação se necessário\n\n### 5. Commit suas Mudanças\n```bash\ngit add .\ngit commit -m \"tipo: descrição da mudança\"\n```\n\n**Tipos de commit:**\n- `feat`: Nova feature\n- `fix`: Bug fix\n- `docs`: Documentação\n- `style`: Formatação\n- `refactor`: Refatoração de código\n- `test`: Testes\n- `chore`: Tarefas gerais\n\n### 6. Push e Pull Request\n```bash\ngit push origin feature/sua-feature\n```\n\nVá para GitHub e crie um Pull Request\n\n## Código de Conduta\n\n- Seja respeitoso\n- Não tolerar discriminação\n- Focar em criatividade e inovação\n\n## Perguntas?\n\nAbra uma issue ou entre em contato conosco!\n\nObrigado por contribuir! ❤️\n"
